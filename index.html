개발 블로그 시작!

Tue, Aug 6, 2019 무슨 말을 해야 할까.. 나만 볼 수 있는 글은 많이 써왔지만, 이렇게 공간을 열어놓고 글을 쓰는 것은 참 어색하다. 한 단어를 쓸 때도 지웠다 썼다를 반복하며 조심스러워 진다. 말을 조심해서 해야 하듯, 글도 신중하게 써야 하는 것이므로, 일단은 옳은 자세라 여겨본다. 자주 쓰면 나아질 것이다. 기본적인 마음가짐 코드 컴플리트 라는 책에서, 일단 작동하는 코드를 만드는 것은 쉽지만, 읽기 쉬운 코드를 만드는 것은 어렵다고 했다. 읽기 쉬운 코드를 만들기 위해서는 이미 작동하는 코드를 다시 들여다보며, 불필요한 부분을 제거하고, 불명확한 부분을 명시하는 등의 작업을 해야한다.
